# Stock Portfolio Tracker (Python)

## Description
This is a console-based Stock Portfolio Tracker developed using Python.
The program allows users to calculate the total value of their stock
investments based on predefined stock prices and user-entered quantities.

This project was created as part of the Python Programming Internship
at CodeAlpha.

## Features
- Predefined stock price list
- User input for stock symbols and quantities
- Calculation of individual stock investment value
- Calculation of total portfolio value
- Input validation for better reliability

## Technologies Used
- Python 3

## How to Run the Program
1. Ensure Python 3 is installed on your system
2. Open a terminal or command prompt
3. Navigate to the project directory
4. Run the following command:

```bash
python stock_portfolio.py
